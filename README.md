# template-react-lib

> react 组件库

[![NPM](https://img.shields.io/npm/v/template-react-lib.svg)](https://www.npmjs.com/package/template-react-lib) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save template-react-lib
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'template-react-lib'
import 'template-react-lib/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
